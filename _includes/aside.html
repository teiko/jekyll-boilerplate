Aside

This skeleton for the simplest of content sites utilises the following components:

  • Jekyll
  • HTML5 Boilerplate
  • SASS
  • jQuery
  • Modernizr
  • Selectivizr
  • Universal IE6 CSS
  • Capistrano for deployment

The following jQuery plugins are bundled:

  • Easing v1.3
  • Live Query v1.03
  • SlidesJS v1.1.6
  • bxSlider v3.0
  • Retina Display v1.2

Deploy using the inbuilt Capistrano script or just upload your generated _site directory to any Linux hosting package.
